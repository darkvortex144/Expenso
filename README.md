# Expenso

Expenso is an offline-first personal finance management app built with:

- Kotlin
- Jetpack Compose
- Room Database
- Material 3

## Features

- Expense tracking
- Income tracking
- Budget management
- Category management
- Account management
- Analytics and charts
- Offline data storage

## Requirements

- Android Studio
- Android SDK 24+

## Build

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on device or emulator
