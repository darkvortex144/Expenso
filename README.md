# Expenso
Expenso is a powerful and easy to use, offline personal expense tracking and money management app.
***
## Features

- Dashboard with financial overview
- Track daily expenses and income
- Manage multiple accounts and wallets
- Set and monitor budgets
- Organize transactions with custom categories
- Visual analytics and spending insights
- Filter transactions by date range
- Search and view transaction history
- Real-time balance calculation
- Modern Material 3 UI with dark mode support
- Secure local data storage using Room Database
- Offline-first experience with no internet required
- Fast and responsive Jetpack Compose interface
- Customizable expense and income categories
- Category-wise spending breakdown
- Financial statements in pdf format
